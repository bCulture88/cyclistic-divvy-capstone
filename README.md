# Cyclistic Bike-Share Case Study

**Google Data Analytics Capstone Project**  
**Author:** Luca Mariani  
**Date:** April 2026

## Project Overview

This repository contains the complete analysis for the **Cyclistic Bike-Share Case Study**, part of the Google Data Analytics Professional Certificate.

The project investigates how **casual riders** and **annual members** use Cyclistic bikes differently, with the goal of providing marketing insights to convert casual riders into annual members.

## Key Findings
- Casual riders take significantly longer rides on average (**19.9 minutes**) vs members (**12.2 minutes**).
- Members use the service more consistently throughout the week, while casual riders strongly peak on weekends.
- Saturday is the busiest day overall with over 839,000 rides.
